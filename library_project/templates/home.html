Library Management System API

This project is running successfully. Use the endpoints below to interact with the API.

Primary Routes

  • /admin/ - Django admin panel
  • /api/books/ - Books endpoints
  • /api/auth/register/ - Register
  • /api/auth/token/ - Token authentication
  • /api/users/ - Users (admin)
  • /api/borrow/checkout/ - Checkout
  • /api/borrow/records/ - Borrow records (admin)
  • /api/return/ - Return
  • /api/my-borrows/ - My borrows

Use Authorization: Token <your_token> for protected endpoints.
